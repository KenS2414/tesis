# Diagramas

Los diagramas generados están disponibles aquí:

- PDF (vector, recomendado para insertar en LaTeX): [diagrams/er_diagram.pdf](diagrams/er_diagram.pdf)
- PNG (raster, 2000px de ancho): [diagrams/er_diagram.png](diagrams/er_diagram.png)

Ejemplo LaTeX para incluir el PDF:

```latex
\begin{figure}[ht]
  \centering
  \includegraphics[width=0.8\textwidth]{diagrams/er_diagram.pdf}
  \caption{Diagrama ER de la base de datos}
  \label{fig:er-diagram}
\end{figure}
```
